Installation
============

The OME Framework runs on Python 2.x and can be installed with pip on linux or OSX

```sudo pip install ome
```

If you are intending to load a lot of models and data, you should also install postgres

On linux

```
sudo apt-get install postgres-server postgres
```

On OSX with homebrew http://brew.sh/

```
brew install postgres
```
